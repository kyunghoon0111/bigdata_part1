3부 · 작업형 1유형 예상문제 (ch3) — 자가테스트

문제를 먼저 풀어보고, 클릭해서 정답 코드를 확인하세요. (데이터: type1_data1.csv 외 배달·학교·판매 데이터)

데이터 컬럼 요약 — type1_data1: id, age, city, f1, f2, f3(silver/gold/vip), f4(MBTI문자), f5, subscribed(날짜), views / delivery_time: 주문시간, 실제도착시간, 예상도착시간, 앱종류, 거리, 결제종류, user / school_data: 이름, 수학, 영어, 국어, ~교사

① 기본 전처리 · 통계 (type1_data1)

Q1. f5≠0 행만 남기고, views 결측을 최솟값으로 채운 뒤 중앙값(정수)?

기법: 조건필터 + fillna(min) + median

cond = df['f5'] != 0
df = df[cond]
df['views'] = df['views'].fillna(df['views'].min())
print(int(df['views'].median()))
Q2. subscribed 최빈값에서 '일'(끝 2자리)만 추출?

기법: value_counts + index + 문자 슬라이싱

vc = df['subscribed'].value_counts()
int(vc.index[0][-2:])   # 최빈값의 마지막 2글자
Q3. views/f1 파생컬럼 내림차순 정렬 후 1위 행의 age?

기법: dropna + 파생컬럼 + sort_values + iloc

df = df.dropna()
df['new'] = df['views'] / df['f1']
df = df.sort_values('new', ascending=False)
int(df.iloc[0, 1])
Q4. views 결측 0으로 채우고, 상위 10개를 그중 최솟값으로 바꾼 뒤 합?

기법: nlargest로 10번째 값 → loc 조건 대치 → sum

df['views'] = df['views'].fillna(0)
value = df['views'].nlargest(10).iloc[-1]  # 10번째로 큰 값
df.loc[df['views'] > value, 'views'] = value
print(int(df['views'].sum()))
Q5. f4의 3~4번째 글자가 'FJ'인 행의 f2 평균(소수 2자리)? (str.contains 버전도)

기법: str 슬라이싱 / str.contains + mean + round

df['new'] = df['f4'].str[2:4]
df = df[df['new'] == 'FJ']
round(df['f2'].mean(), 2)
# 또는: df[df['f4'].str.contains('FJ')]['f2'].mean()
Q6. f3=='gold' AND f2==2인 행의 f1 분산(소수 2자리)?

기법: 다중조건(&) + var

cond1 = df['f3'] == 'gold'
cond2 = df['f2'] == 2
df = df[cond1 & cond2]
round(df['f1'].var(), 2)
Q7. age에 +1 후, 20대와 30대의 views 평균 차이(절댓값)?

기법: 구간조건 + mean 비교 + abs

df['age'] = df['age'] + 1
cond1 = (df['age']>=20) & (df['age']<30)
cond2 = (df['age']>=30) & (df['age']<40)
round(abs(df[cond1]['views'].mean() - df[cond2]['views'].mean()), 2)
Q8. subscribed를 날짜로 변환, 2024년 2월 & f3=='gold'인 행 개수?

기법: to_datetime + dt.year/month + 다중조건 + len

Q12. 결측 최다 컬럼의 결측행 제거 후, f3 결측을 최빈값으로 채우고 'gold' 개수?

기법: isnull().sum().idxmax() + dropna(subset) + fillna(mode)

print(len(df[df['f3']=='gold']))   # sum(df['f3']=='gold')도 가능

5부 · 자주 나오는 고급 패턴 (암기 레시피)

작업형1에서 반복되는 패턴을 코드 레시피로 압축. 자가테스트가 막히면 여기서 골격을 찾으세요.

1. IQR 이상치 탐지

q1 = df['x'].quantile(.25)
q3 = df['x'].quantile(.75)
IQR = q3 - q1
low  = q1 - 1.5*IQR     # 아래 경계
high = q3 + 1.5*IQR     # 위 경계
이상치 = df[(df['x']<low) | (df['x']>high)]
정상치 = df[(df['x']>=low) & (df['x']<=high)]

2. 스케일링 (Min-Max / 표준화)

# Min-Max: 0~1로 압축
df['mm'] = (df['x'] - df['x'].min()) / (df['x'].max() - df['x'].min())
# 표준화(Z-score): 평균0 표준편차1
df['z'] = (df['x'] - df['x'].mean()) / df['x'].std()

3. datetime 시간차

df['a'] = pd.to_datetime(df['a'])
df['b'] = pd.to_datetime(df['b'])
diff = df['b'] - df['a']            # Timedelta
diff.dt.total_seconds()/60       # 분
diff.dt.total_seconds()/3600     # 시간
diff.dt.days                       # 일수
df['a'].dt.hour                    # 시간대 필터용

4. groupby.transform (그룹값을 원본 행에)

# 그룹 평균으로 결측 채우기
m = df.groupby('g')['x'].transform('mean')
df['x'] = df['x'].fillna(m)
# 그룹 평균과의 편차
df['편차'] = df['x'] - df.groupby('g')['x'].transform('mean')

transform은 원본과 길이가 같아 새 컬럼으로 바로 대입 가능. agg는 그룹당 1행으로 줄어듦.
